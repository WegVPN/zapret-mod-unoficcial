# Resources Folder

This folder should contain the application icon file.

## Required Files

### zapret.ico

Create an icon file with the following specifications:
- **Format**: ICO (Windows icon)
- **Sizes included**: 16x16, 32x32, 48x48, 64x64, 128x128, 256x256
- **Color depth**: 32-bit with alpha channel

## Icon Design Suggestions

The icon should represent:
- Network/Internet connectivity
- Security/Privacy (shield, lock)
- Speed/Optimization (lightning bolt, arrow)

### Color Scheme
- Primary: Blue (#0078D4) - represents trust and technology
- Secondary: Green (#107C10) - represents success and connectivity
- Accent: Orange (#FF8C00) - represents speed and optimization

## Creating the Icon

You can create the icon using:
1. **Visual Studio**: Add new item → Icon File
2. **Online tools**: 
   - https://www.icoconverter.com/
   - https://convertio.co/png-ico/
3. **Graphics software**: GIMP, Photoshop with ICO plugin

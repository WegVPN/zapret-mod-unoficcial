# 🛡 ZapretMod - DPI Bypass для Discord, YouTube, Telegram

## 🚀 Быстрый старт

### Шаг 1: Скачайте бинарные файлы

**Вариант A (Автоматически):**
1. Запустите `download-binaries.bat`
2. Дождитесь окончания загрузки

**Вариант B (Вручную):**
1. Скачайте с https://github.com/bol-van/zapret-win-bundle/releases/latest
2. Распакуйте файлы в папку `bin\`:
   - `winws.exe`
   - `WinDivert64.sys`
   - `WinDivert64.dll`

### Шаг 2: Запуск

1. Запустите `ZapretMod.exe` **от имени администратора**
2. Выберите стратегию (рекомендуется "Discord + YouTube + Telegram")
3. Нажмите **"▶ ЗАПУСТИТЬ"**

## 📋 Стратегии

| Стратегия | Описание |
|-----------|----------|
| **Discord + YouTube + Telegram** | Основная стратегия для всех сервисов |
| **Discord Only** | Только Discord (включая голосовые каналы) |
| **YouTube Only** | Только YouTube |
| **Telegram Only** | Только Telegram |
| **FAKE TLS AUTO** | Автоматический fake TLS |
| **SIMPLE FAKE** | Простой fake метод |

## ⚙️ Требования

- Windows 10/11 x64
- .NET 8 Desktop Runtime
- Права администратора

## 🔍 Диагностика

Если не работает:

1. Откройте вкладку **"🔍 Диагностика"**
2. Проверьте наличие всех файлов
3. Убедитесь что запущены от администратора
4. Включите Secure DNS
